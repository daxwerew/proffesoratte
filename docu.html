Como Funciona

Esta es una pequeña ayuda de como interactuar con el esqueleto MVC del proyecto proffesoratte, para llegar a un controlador se debe acceder a website/?ctrl=TuControlador

A continuacion todos los controladores y sus posibles recepeciones de datos, por ahora todas son por method Get

Hosting: 000webhost.com
Sitio: http://proffesoratte.freeiz.com/

Controladores

Alumnos

Nombre Archivo: alumnosCtrl.php

Acciones

Alta

Crear Alumno

  • Url: website/?ctrl=alumnos&accion=alta
  • Parametros Forzosos:
    • nombre: que cumpla /^[a-z| ]+$/i
    • codigo
    • carrera
    • email: que cumpla /^[A-Z0-9._%+-]+@[A-Z0-9.-]+\.[A-Z]{2,4}$/i
  • Ejemplo: website/?ctrl=alumnos&accion=alta&nombre=Diego&codigo=20&carrera=COM&email=a@a.net

Baja

Eliminar Alumno

  • Url: website/?ctrl=alumnos&accion=baja
  • Parametros Forzosos:
    • codigo
  • Ejemplo: website/?ctrl=alumnos&accion=baja&codigo=20

Consulta

Consulta datos de un Alumno

  • Url: website/?ctrl=alumnos&accion=consulta
  • Parametros Forzosos:
    • codigo
  • Ejemplo: website/?ctrl=alumnos&accion=consulta&codigo=20

Modificar

Consulta datos de un Alumno

  • Url: website/?ctrl=alumnos&accion=modificar
  • Parametros Forzosos:
    • codigo
  • Parametros Opcionales:
    • nombre: que cumpla /^[a-z| ]+$/i
    • carrera
    • email: que cumpla /^[A-Z0-9._%+-]+@[A-Z0-9.-]+\.[A-Z]{2,4}$/i
  • Ejemplo: website/?ctrl=alumnos&accion=modificar&nombre=Diego&codigo=20&carrera=COM&email=a@a.net

Ciclo Escolar

Nombre Archivo: cicloEscolarCtrl.php

Acciones

Alta

Crear ciclo escolar y sus dias apartir de una fecha de inicio y fecha de fin

  • Url: website/?ctrl=cicloEscolar&accion=alta
  • Parametros Forzosos:
    • ciclo: que coincida con /^2[0-9]{3}[ABV]$/i
    • fechaInicio
    • fechaFinal
    • diasSemana: un arreglo de dias de semana donde 0 es domingo y 6 es sabado
  • Ejemplo: website/?ctrl=cicloEscolar&accion=alta&ciclo=2014A&fechaInicio=14/02/2014&fechaFinal=14/06/2014&diasSemana[]=1&diasSemana[]=3

Modificar

Modificar ciclo escolar

  • Url: website/?ctrl=cicloEscolar&accion=modificar
  • Parametros Forzosos:
    • ciclo
    • festivos: arreglo de fechas a excluir
  • Ejemplo: website/?ctrl=cicloEscolar&accion=modificar&ciclo=2014A&festivos[]=21/03/2014&festivos[]=14/04/2014&

Baja

Elimina ciclo escolar

  • Url: website/?ctrl=cicloEscolar&accion=baja
  • Parametros Forzosos:
    • ciclo
  • Ejemplo: website/?ctrl=cicloEscolar&accion=baja&ciclo=2014A

Consulta

Consulta ciclo escolar

  • Url: website/?ctrl=cicloEscolar&accion=consulta
  • Parametros Forzosos:
    • ciclo
  • Ejemplo: website/?ctrl=cicloEscolar&accion=consulta&ciclo=2014A

Curso

Nombre Archivo: cursoCtrl.php

Acciones

Alta

Crea un curso

  • Url: website/?ctrl=curso&accion=alta
  • Parametros Forzosos:
    • cicloEscolar
    • nombre
    • seccion: que cumpla /^([a-z]{2}\d{3}|[a-z]?\d{4})-[a-z]\d{2}$/i, Ejem AB444-Z99
    • academia
    • nrc
    • diasSemana_Horas: arreglo donde el indice sea el dia de la semana (0 para domingo y 6 para sabado) y el valor sea las horas de ese dia, Ejem. arre[1]=3 seria equivalente a decir lunes=3 horas
    • fechaInicio
    • fechaFinal
  • Ejemplo: website/?ctrl=curso&accion=alta&cicloEscolar=2014A&nombre=Compiladores&seccion=CC100-D02&academia=COM&nrc=80321&diasSemana_Horas[1]=3&diasSemana_Horas[5]=4&fechaInicio=14/02/2014&fechaFinal=20/07/2014

Baja

Elimina curso

  • Url: website/?ctrl=curso&accion=baja
  • Parametros Forzosos:
    • seccion: que cumpla /^([a-z]{2}\d{3}|[a-z]?\d{4})-[a-z]\d{2}$/i, Ejem AB444-Z99
  • Ejemplo: website/?ctrl=curso&accion=baja&seccion=CC100-D02

Consulta

Consulta curso

  • Url: website/?ctrl=curso&accion=consulta
  • Parametros Forzosos:
    • seccion: que cumpla /^([a-z]{2}\d{3}|[a-z]?\d{4})-[a-z]\d{2}$/i, Ejem AB444-Z99
  • Ejemplo: website/?ctrl=curso&accion=consulta&seccion=CC100-D02

Listas

Nombre Archivo: listasCtrl.php

Acciones

Alta

Crea una Lista

  • Url: website/?ctrl=listas&accion=alta
  • Parametros Forzosos:
    • alumnos: arreglo de alumnos
    • curso: existente
    • cicloEscolar: existente
  • Ejemplo: website/?ctrl=listas&accion=alta&alumnos[]=pepe&alumnos[]=diego&curso=CC100-D05&cicloEscolar=2014A

Modificar

Modifica una Lista

  • Url: website/?ctrl=listas&accion=modificar
  • Parametros Forzosos:
    • alumnos: arreglo de alumnos existentes
    • curso: existente
    • cicloEscolar: existente
    • lista: existente
  • Ejemplo: website/?ctrl=listas&accion=modificar&alumnos[]=pepe&alumnos[]=diego&curso=CC100-D05&cicloEscolar=2014A&lista=14

Baja

Elimina Lista

  • Url: website/?ctrl=listas&accion=baja
  • Parametros Forzosos:
    • lista: existente
  • Ejemplo: website/?ctrl=listas&accion=baja&lista=14

Consulta

Consulta Lista

  • Url: website/?ctrl=listas&accion=consulta
  • Parametros Forzosos:
    • lista: existente
  • Ejemplo: website/?ctrl=listas&accion=consulta&lista=14
